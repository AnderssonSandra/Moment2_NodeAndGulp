Moment 2- Node och Gulp

Det här är en uppgift i kursen webbutveckling 3 som jag läser på programmet webbutvecklare på Mittuniversitetet. Uppgiften går ut på att lära sig att använda Node.js och Gulp. Vi jobbar med att automatisera olika delar i utvecklingen för att underlätta för oss utvecklare. I den här uppgiften skapar vi en grundstruktur för att automatisera olika uppgifter för att slå ihop, minifiera och konkatenera CSS, HTML, Javascript samt bilder i olika format.
